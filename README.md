# V聚焦

纯Flutter构建的一款跨平台app，主要功能包含:

    1.用户登录&注册&找回密码，登陆界面采用了精美,大气,漂亮的2D动画
    2.RSS栏目订阅&RSS列表&详情，支持收藏
    3.多主题切换，颜色随心所欲
    4.[https://www.lanzous.com/i56dj0d](更多请自行下载体验,点我)
